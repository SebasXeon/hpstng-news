# Hijueposting News

Generador de imagenes estilo noticia. Sube una imagen, escribe un titular y descarga el resultado.

### Uso

Abre `index.html` en el navegador.

### BBCode

- `[L]` / `[M]` / `[S]` al inicio de linea para tamano grande, mediano o pequeno
- `[rojo]texto[/rojo]` / `[azul]texto[/azul]` para colorear palabras
